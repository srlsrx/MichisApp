Module: Header

Componente Header que incluye un botón de menú responsive para abrir/cerrar el Sidebar.

Este componente muestra una barra de navegación superior con un logo, enlaces de navegación estilizados y un botón de menú para dispositivos móviles y tabletas. Los enlaces se adaptan a la ruta actual para evitar anidamientos incorrectos.
