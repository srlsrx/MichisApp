Module: Sidebar

Componente Sidebar que se desliza para mostrar un menú de navegación.

Este componente muestra un menú lateral que se puede abrir y cerrar en dispositivos móviles y tablets. Se activa desde el Header y permite acceder a diferentes secciones de la aplicación con enlaces estilizados.
